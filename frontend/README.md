# Glam Cart Builder - 化妆品包装购物车系统

## 项目概述

这是一个专业的化妆品包装购物车系统，专为化妆品包装供应商设计。用户可以浏览各种化妆品包装产品，添加到购物车，并生成专业的报价单。

## 主要功能

### 🛍️ 产品浏览与筛选
- 丰富的化妆品包装产品库
- 智能筛选功能（材质、形状、类型等）
- 产品详情展示（规格、价格、图片）

### 🛒 购物车功能
- 添加产品到购物车
- 数量调整和删除功能
- 实时价格计算

### 📋 专业报价单生成
- **全新简约设计** - 采用现代极简风格
- **清晰信息层次** - 专业的排版和字体层次
- **高级配色方案** - 黑白灰为主的经典配色
- **用户友好体验** - 清晰的产品明细和价格展示
- **多种分享方式** - 支持保存、下载和分享

### 🌐 多语言支持
- 中文/英文界面切换
- 完整的国际化支持

### ⚡ 响应式设计
- 完美适配桌面端和移动端
- 流畅的用户交互体验

## 报价单设计特色

### 🎨 现代简约风格
- **配色方案**: 采用黑白灰经典配色，专业大气
- **字体设计**: 使用 Inter 和 SF Mono 字体，提升可读性
- **布局结构**: 清晰的信息层次，重要信息突出显示

### 📱 用户体验优化
- **信息组织**: 产品明细、数量、价格一目了然
- **视觉层次**: 通过字体大小、颜色深浅建立清晰层次
- **空间利用**: 合理的间距和留白，提升视觉舒适度

### 💼 专业性提升
- **格式规范**: 标准的商业报价单格式
- **信息完整**: 包含报价单号、日期、产品详情、总计等
- **法律声明**: 明确的价格说明和法律免责声明

## 技术架构

### 前端技术栈
- **React 18** - 现代化的用户界面框架
- **TypeScript** - 类型安全的开发体验
- **Vite** - 快速的构建工具
- **Tailwind CSS** - 原子化CSS框架
- **shadcn/ui** - 高质量的组件库

### 特色功能实现
- **Canvas图片生成** - 将报价单转换为图片格式
- **响应式设计** - 使用CSS Grid和Flexbox
- **状态管理** - React Hooks进行状态管理
- **国际化** - 完整的多语言支持系统

## 开发指南

### 环境要求
- Node.js 16+ 
- npm 或 yarn

### 安装步骤

```bash
# 克隆项目
git clone <YOUR_GIT_URL>

# 进入项目目录
cd glam-cart-builder

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
